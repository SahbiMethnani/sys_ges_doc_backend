NOTES DE VERSION - HISTORIQUE DES MISES À JOUR
Système de Gestion Documentaire

=================================================================
VERSION 2.4.0 - Mars 2026
=================================================================

NOUVELLES FONCTIONNALITÉS
--------------------------
- Système RAG (Retrieval-Augmented Generation) : posez des questions en langage naturel sur vos documents
- Recherche sémantique : trouvez des documents par leur sens, pas seulement par mots-clés
- Assistant IA intégré : génération automatique de résumés de documents
- Support des fichiers RAW et formats texte brut amélioré

AMÉLIORATIONS
-------------
- Performance de l'upload améliorée de 40% pour les fichiers de grande taille
- Interface utilisateur modernisée (nouveau thème sombre disponible)
- Notifications en temps réel via WebSocket (plus de délai)
- API v2 : nouveaux endpoints pour la gestion des webhooks

CORRECTIONS DE BUGS
-------------------
- Correction de l'affichage des PDFs protégés par mot de passe
- Résolution du problème de déconnexion intempestive après 30 minutes
- Correction de l'export ZIP qui échouait pour les dossiers de plus de 1000 fichiers

NOUVELLES FONCTIONNALITÉS

CORRECTIONS DE BUGS
-------------------

- Résolution des problèmes de téléchargement sur Safari iOS 16

VERSION 2.0.0 - Mars 2025
=================================================================

REFONTE MAJEURE
---------------
Cette version marque une refonte complète du système.

NOUVELLES FONCTIONNALITÉS
--------------------------
- Architecture microservices pour une meilleure scalabilité
- Nouveau modèle de permissions RBAC (rôles et droits granulaires)
- API REST complète avec documentation OpenAPI/Swagger
- Support des webhooks
- Intégration Slack
- Authentification 2FA via TOTP

MIGRATION DEPUIS V1
-------------------
La migration depuis la version 1.x est automatique.
Toutes vos données ont été conservées.
Les anciennes URLs de l'API v1 restent fonctionnelles jusqu'au 31 Décembre 2025.
